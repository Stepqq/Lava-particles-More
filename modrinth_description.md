**Lava Particles | More** is a client-side mod that completely changes how Minecraft lava feels. It adds dynamic bubbles, sizzling effects, volcanic geysers, and splash physics to make caves and the Nether look much more immersive.

## ✨ Features

### 🌋 Lava Geysers & Eruptions
* **Magma Geysers**: Large lava pools occasionally erupt with rising columns of heat-distortion waves, bubbles, and hot flying droplets.
* **Ambient Sounds**: Eruptions are accompanied by deep, low-frequency volcanic rumbling.

### 🫧 Bubbling Effects
* **Surface Bubbles**: Lava pools bubble actively with tiny fizzing pops on the surface.
* **Deep Magma Bubbles**: Large magma bubbles rise slowly from the depths of lava lakes and burst at the surface.

### ☄️ Splash & Droplet Physics
* **Landing Droplets**: Flying magma drops land on blocks, slide around, and slowly cool down (changing color from bright red to dark basalt) before disappearing.
* **Water Interactions**: Droplets fizz and sizzle when landing in water, creating steam and rising water bubbles.
* **Droplet Damage (Optional)**: Magma drops can deal minor fire damage (`0.5 hearts`) to players, with a customizable cooldown.

### 💨 Realistic Combustion
* **Lava Sizzling**: Submerging or walking in lava creates dense columns of smoke, sparks, bubbling, and constant sizzling sound effects.

## 🛠️ Installation & Compatibility

> [!NOTE]
> **This mod is purely client-side!** You do not need to install it on servers to enjoy the visual atmospheric improvements in multiplayer.

* **Supported Loaders**: Fabric, Forge, NeoForge
* **Supported Versions**: Minecraft `1.20.1` & `1.21.1`
* **Dependencies**: 
  - **Fabric**: Requires [Fabric API](https://modrinth.com/mod/fabric-api) and [Cloth Config API](https://modrinth.com/mod/cloth-config) for configuration GUI.
  - **Forge / NeoForge**: Requires [Cloth Config API](https://modrinth.com/mod/cloth-config).

## ⚙️ Configuration Options

You can customize everything in-game using the configuration menu:

| Tab | Option Name | Description | Default Value |
| :--- | :--- | :--- | :--- |
| **Visuals** | **Check Frequencies** | Adjust update rates to optimize performance. | `1.0` |
| | **Particle Spawn Multipliers** | Control the density of lava and smoke particles. | `1.0` |
| | **Overworld Activity Multipliers** | Adjust eruption frequency outside the Nether. | `0.5` |
| | **Magma Floor Stay Time** | How long droplets stay on the ground before cooling down. | `3.0s` |
| | **Large Bubble Toggles** | Enable or disable large bubbling effects. | `true` |
| **Audio** | **Mod Volume Multiplier** | Master volume control for custom sounds. | `1.0` |
| | **Pop Sound Toggle** | Toggle the sound of small bursting bubbles. | `true` |
| | **Rumble Sound Toggle** | Toggle deep volcanic rumbles. | `true` |
| **Gameplay** | **Player Lava Sizzling** | Toggle steam and sizzling effects when in lava. | `true` |
| | **Droplet Fire Damage** | Toggle whether magma droplets deal minor fire damage. | `true` |
| | **Damage Cooldown** | Cooldown time between droplet damage ticks. | `3.0s` |
